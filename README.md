# UPI-Transactions-Data-Analysis (Power BI)


📌 Project Overview
This project focuses on analyzing UPI (Unified Payments Interface) transaction data using Power BI to uncover meaningful insights such as transaction trends, user behavior, and financial patterns.
The dataset is sourced from an Excel file, and the report is built using Power BI with interactive dashboards and multiple visualizations.


🛠️ Tools & Technologies
Power BI
Microsoft Excel (Data Source)
Power Query (Data Cleaning & Transformation)
DAX (Data Analysis Expressions)


📂 Data Source
The dataset contains UPI transaction information with the following fields:
BankNameSent
BankNameReceived
City
DeviceType
Gender
Age Groups
Merchant Name
Payment Method
Purpose
Transaction Type
Transaction Amount
Remaining Balance
Month


📈 Dashboard Features
✅ 1. Interactive Filters (Slicers)
Bank Name (Sent & Received)
City
Device Type
Gender
Age Groups
Merchant Name
Payment Method
Purpose
Transaction Type
These slicers enable dynamic exploration of the data.


✅ 2. Monthly Transaction Analysis
Line Chart showing trends across months
Example observations:
Highest transactions around May (~1.70M)
Lowest point around August (~1.59M)
Visible fluctuations throughout the year


✅ 3. Column Chart Visualization
Monthly transaction comparison using column chart
Helps identify trends and peak periods clearly
Stable trend observed towards year-end


✅ 4. Matrix Visual (City-wise Analysis)
Displays:
City-based transactions (Bangalore, Delhi, Hyderabad, Mumbai)
Currency variations (EUR, USD, GBP, INR)
Monthly amounts and remaining balances
Useful for comparing financial activity across locations


✅ 5. Multi-Page Report
Page 1: Monthly trends (Line & Column charts)
Page 2: City-wise detailed matrix and balances


✅ 6. Bookmarks & Navigation
Implemented bookmarks to:
Toggle between Line Chart and Column Chart
Switch between Transactions and Remaining Balance views
